1_2 最终代码和效果 #

本节将分享一下最终代码和效果。

最终代码 #

  • 服务器只有一个index.js文件，内容如下：
const Koa = require('koa');
const KoaStatic = require('koa-static');
const cp = require('child_process');

const app = new Koa();

// 使用koa-static中间件处理静态资源
app.use(KoaStatic('./public')); 

app.listen(3600);

// 在非生产环境下，自动打开浏览器访问服务
cp.exec('open http://localhost:3600/');
